Phonogeometry

Sentences as trajectories in a bounded articulatory feature space. A pipeline, not a metaphor.

Thesis in one paragraph. Map graphemes to phonemes (via dictionary IPA), embed each phoneme as a point in a phonetic feature manifold (bounded, interpretable, 2D here but extensible), and treat a sentence as the resulting polyline — a discrete articulatory kinematic trajectory (AKT [Chartier et al. 2018]) in abstract space rather than physical vocal-tract space. Many linguistic operations then become computational-geometry queries on a lattice-valued polyline: rhyme = suffix nearest-neighbor, alliteration = prefix clustering, tongue-twister difficulty = curvature density, sentence compression = Ramer–Douglas–Peucker simplification keeping only articulatorily salient phonemes, phonetic efficiency = straight-line / path length. The mapping is a bijection on phonemes (lossless), so this sits alongside existing IPA pipelines rather than replacing them.

1 · Pipeline

grapheme string → G2P (IPA via dictionaryapi.dev + fallback) → phoneme tokenization → articulatory feature vectors → 2D projection → polyline → analysis

The articulatory space is a subset of PanPhon-style features [Mortensen et al. 2016] collapsed to two interpretable axes for visualization:

  • X axis: front-to-back — vowels use backness, consonants use place of articulation (bilabial → labiodental → dental → alveolar → postalveolar → palatal → velar → glottal).
  • Y axis: openness / sonority — close vowels at the top, open vowels mid, then consonants stacked by manner (glides → liquids → nasals → fricatives → affricates → stops).

This is a projection — the real space is ~7-dimensional. The 2D layout is chosen to preserve the sonority hierarchy vertically and place horizontally, giving visually coherent trajectories (rising sonority arcs up, consonant clusters stay low).

3 · What the numbers mean

Curvature density (tongue-twister score). We sum unsigned turning angles along the polyline, normalize by phoneme count. Tongue twisters — sentences that repeatedly shuttle between nearby but distinct places of articulation (s/ʃ, p/b, k/g) — produce high-angle zigzags. This matches the clinical characterization: rapid reversals between minimal-pair consonants [PANCETTA, Keh et al. 2022].
Path efficiency η = ‖pn − p1‖ / ∑‖pi+1 − pi‖. 1.0 means the sentence marches directly across the mouth; 0 means we end where we started after a meandering tour. Alliterative/rhyming verse has structurally low η — the path loops back on itself.
RDP-compressed sentence. Ramer–Douglas–Peucker [Ramer 1972; Douglas & Peucker 1973] keeps only phonemes whose perpendicular distance from the chord of their neighbours exceeds ε. What survives are the phonemes at genuine direction changes in the articulatory trajectory — a principled analogue of "acoustic landmarks" [Stevens 2002], computable in O(n log m) on average. This is different from prior AKT compression work, which operates on continuous physical signals; here we operate on the discrete feature-space trajectory.
Rhyme pairs. For each word pair in the sentence, compute the 2-norm between the last-k phoneme vectors (k=2). Pairs below threshold = rhyme. Zero threshold = perfect rhyme; ε-ball = slant rhyme. Free.
Alliteration runs. Consecutive words whose initial phonemes fall within ε of each other → alliterative run. This captures true phonetic alliteration (soft/city share /s/) even when spelling disagrees.

5 · Metric properties (a PhD committee would ask)

  • Non-negativity, identity, symmetry: satisfied — we use Euclidean in feature space.
  • Triangle inequality: satisfied on pointwise distances; not satisfied by DTW between unequal-length curves. This is a known property of DTW [Casacuberta 1987].
  • Perceptual calibration: coords are hand-tuned, not learned from human similarity judgments. A real thesis would fit a metric against MDS on confusion matrices [Miller & Nicely 1955].
  • Information bound: with |Σ| ≈ 40 English phonemes, ceiling entropy is log₂ 40 ≈ 5.3 bits/phoneme. English phoneme bigram entropy ≈ 3.1 bits (Shannon-style estimate). RDP-compressed streams trade lossiness for lower bitrate — the rate-distortion curve is an empirical question this tool can generate.

6 · Honest limitations

This is a working sketch, not a thesis. Concretely missing: (a) suprasegmental features — no stress, tone, duration, intonation; (b) the 2D projection is lossy vs. the 7+ dim real feature space; (c) feature weighting is uniform but human perception isn't (Weber's law, phonotactic priors); (d) G2P fallback is a toy; (e) coordinates are English-biased (no clicks, ejectives, retroflex series); (f) we don't handle coarticulation, which AKT research shows is huge; (g) dictionaryapi.dev sometimes fails CORS or returns no IPA — the app falls back to a local mini-dictionary then a last-ditch letter-based approximation; (h) no empirical benchmark against PWESuite, TwistList, or CMU-dict rhyme gold.

7 · Open questions a real thesis would tackle

  1. Learn coordinates by metric learning on human phonetic-similarity judgments (Nosofsky-style MDS) and compare to hand-tuned.
  2. Empirically test: does RDP-compressed speech remain intelligible? Design a synthesis experiment — resynthesize only the kept phonemes with schwa insertions, run a WER study.
  3. Benchmark: does trajectory-DTW beat Soundex / Metaphone / pretrained phonetic embeddings on PWESuite's rhyme + cognate tasks?
  4. Cross-linguistic: the PanPhon feature set is universal. Does the sonority-based Y axis generalize, or do we need language-specific projections?
  5. Hybrid semantic-phonetic space: concatenate our 2–7 dim articulatory features with a small semantic vector (FastText subword). Does that beat either alone on pun / homophone detection?
  6. Rate-distortion curve: plot ε vs. intelligibility and vs. bitrate. Is there a phase transition?
  7. Is curvature density a predictor of child-acquisition order (later-acquired phonemes cluster closer, so twisters using late phonemes score higher)?
  8. The AKT work shows coarticulation is locally encoded. Can we predict coarticulation from 2nd-derivative smoothing of our trajectory?
